# Calculadora de Tinta para Paredes

Um script simples em Python para calcular a quantidade de latas de tinta necessárias para pintar uma determinada área e o custo total.

## 🚀 Como usar

1.  **Clone o repositório (se estiver no GitHub):**
    ```bash
    git clone [https://github.com/seu-usuario/calculadora-tinta.git](https://github.com/seu-usuario/calculadora-tinta.git)
    cd calculadora-tinta
    ```
    *(Se não estiver no GitHub, apenas salve o arquivo `calculadora_tinta.py` em uma pasta.)*

2.  **Execute o script:**
    ```bash
    python calculadora_tinta.py
    ```

3.  **Insira a área em metros quadrados (m²)** quando solicitado pelo programa.

O programa irá então exibir:
* O número de latas de tinta necessárias (arredondado para cima).
* O custo total em Reais (R$).

## 🛠️ Tecnologias Utilizadas

* **Python 3**

## ✨ Funcionalidades

* Calcula os litros de tinta necessários com base na área fornecida (considerando um rendimento de 1 litro para 3 m²).
* Calcula o número de latas de tinta (cada lata contém 18 litros).
* Arredonda para cima o número de latas para garantir cobertura total.
* Calcula o custo total, considerando o preço de R$120,00 por lata.
